# NetPair — Always Read Master Plan Before Any Work

Before making ANY changes to this project, read `MASTER_PLAN.md` fully and follow it.

**Non-negotiable rules:**
- Never change existing styles, CSS files, Tailwind classes, sidebar, header, or routing
- Only add features inside existing page wrappers using the established UI patterns
- Follow the Global UI Patterns table in Section 3 for every new element
- Follow the Implementation Order in Section 15
- All new modules must follow the page wrapper pattern and button/card/modal standards
